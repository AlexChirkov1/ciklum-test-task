## Installation

Use the package manager [npm](https://docs.npmjs.com/cli/v6/commands/npm-install) to install node_modules.

```bash
npm i
```

## Quick start
**Launch local project** 
```npm
npm start
```
**Build product mode** 
```npm
npm run build
```
**Deploy to github pages** 
```npm
npm rub deploy
```
